3 Usage

Installation

Install the Software Evolution library in the same way as any other Common Lisp package. The Quicklisp (see http://www.quicklisp.org/) package manager may be used to easily install this libraries and all dependencies.

Requires the following Common Lisp packages which are all installable through Quicklisp.

  • alexandria
  • metabang-bind
  • curry-compose-reader-macros
  • split-sequence
  • cl-ppcre
  • elf

Depending on the type of software object used the following software may also be required.

  • llvm-mutate for llvm object support (available from https://github.com/eschulte/llvm-mutate).
  • cil-mutate for cil object support (available from https://github.com/eschulte/cil-mutate).
  • clang-mutate for clang object support (available from https://github.com/eschulte/cil-mutate).

Packaging

The Software Evolution library is meant to be used from Common Lisp programs. Such packages should include both the software-evolution library and may also include the software-evolution-utility library in their package definition as in the following example. 

(defpackage :software-evolution-example
  (:use :common-lisp
        :software-evolution
        :software-evolution-utility))
